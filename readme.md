1. Menu Tambah (Add Data)

- Tambah Developer
  Pilih sub-menu 1. Tambah Developer.
  Masukkan Nama, ID, Negara, dan Tahun.
  Pilih Posisi menyimpan data:
  1 (First): Data masuk ke urutan paling atas.
  3 (Last): Data masuk ke urutan paling bawah.
  2 (After): Program akan meminta ID Developer Sebelumnya sebagai acuan posisi.

B. Tambah Game
Pilih sub-menu 2. Tambah Game.
PENTING: Masukkan ID Developer Pemilik (Parent) tempat game ini bernaung.
Jika Developer ditemukan, masukkan detail game (Nama, ID, Genre, Tahun, Harga).
Pilih posisi penyisipan (1. First / 2. After / 3. Last).

C. Tambah Review
Pilih sub-menu 3. Tambah Review.
Masukkan ID Developer terlebih dahulu, lalu masukkan ID Game.
Masukkan data review (User, Rating 1-5, Komentar).
Pilih posisi penyisipan.

2. Menu Tampilkan (View Data)
   Pilih menu 2 di Menu Utama.
   Gunakan menu ini untuk memverifikasi apakah data yang Anda tambah/hapus/ubah sudah berhasil diproses.

Tampilkan Semua: Menampilkan Data Lengkap.
Tampilkan Developer: Hanya menampilkan list developer.
Tampilkan Game: Program akan meminta Nama Developer, lalu menampilkan game buatan developer tersebut.
Tampilkan Review: Program meminta Nama Developer & Nama Game, lalu menampilkan review spesifik.

3. Menu Update ( Ubah Data )
   Pilih jenis data yang ingin diubah (Developer/Game/Review).
   Masukkan ID Target (Data lama yang ingin diedit).
   Program akan menampilkan kolom isian baru satu per satu.
   Ketik data baru lalu ENTER jika ingin mengubahnya.
   Langsung tekan ENTER (biarkan kosong) jika data tersebut tidak ingin diubah (tetap menggunakan data lama).

4. Menu Delete (Hapus Data)
   Pilih menu 4 di Menu Utama.
   Hati-hati: Menghapus Developer akan menghapus semua Game & Review di dalamnya (Cascading Delete).
   Pilih jenis data yang ingin dihapus.
   Program akan menampilkan opsi penghapusan berdasarkan posisi:

- Delete First: Menghapus data paling atas.
- Delete After: Menghapus data di tengah. Syarat: Anda harus memasukkan ID/Nama SEBELUM data yang ingin dihapus.
- Delete Last: Menghapus data paling bawah.

5. Menu Sortir (Pengurutan)
   Pilih menu 5 di Menu Utama.
   Pilih data yang ingin diurutkan (Developer/Game/Review).
   Untuk Developer: ketik nama, id, atau tahun.
   Untuk Game: ketik nama, id, harga, atau rating.
   Untuk Review: Id Developer, Id Game
   Ketik arah pengurutan:
   asc (Ascending / Kecil ke Besar / A-Z).
   desc (Descending / Besar ke Kecil / Z-A).

6. Menu Searching
   Pilih menu 6 di Menu Utama.
   A. Mencari (Search)
   Pilih opsi pencarian (Developer/Game).
   Masukkan Nama atau ID yang dicari. Hasil akan menampilkan detail data tersebut.

6.1 Menu Filter Game di Searching
Anda akan diberikan beberapa kolom isian (ID Dev, Nama, Genre, Range Harga, Range Rating).
Cara Pakai: Isi kriteria yang diinginkan, dan tekan ENTER (kosongkan) pada kriteria yang tidak diisikan.
Contoh: Jika ingin mencari game dengan harga di bawah Rp 100.000 tapi genrenya bebas, cukup isi bagian "Harga Maksimum" dan kosongkan sisanya.
